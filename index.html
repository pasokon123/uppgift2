Frågor F4. Layout

Förklara vad CSS-egenskaperna justify-content respektive align-items har för effekt när man använder flexbox för layout.

Både justify-content och align-items egenskaperna skjusterar positionen av ett objekt. När justify-content används så flyttas objektet på huvudaxeln och när align-items används så flyttas objektet på den sekundära axeln, axeln vinkelrät mot huvudaxeln. Huvudaxeln kan vara horisontell eller vertikal beroende på vilket värde på flex-direction som används.

Vad är en media query och vad används den till?

En media query används för att ändra CSS egenskaper bara när ett villkor är sant. Används för att göra en webbsida responsiv, till exempel ändra CSS egenskaper för att anpassa webbsidan för olika stora skärmar.
